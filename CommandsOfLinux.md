# Commands of Linux

This is a note about Commands used in Linux

## Quick look
```
###### copy and move   ############
# copy directory and all its files
cp -avr [source] [destination]

# cp the files and subdirectory 
cp -v [source]/* [destination]

# move 
mv [source] [destination]

####### unzip and zip   ##########

####### path     ##############
# return home path
cd ~

# show current path
pwd

# read txt
head [path]

```
